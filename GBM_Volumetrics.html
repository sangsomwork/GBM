Extracting Volumetric Features of Glioblastoma¶

Volumetric features including Contrast Enhancing (CE) region and Necrotic (NE) regions of Glioblastoma (GBM) have strong associations with biological pathways and patient outcomes [1]. In the take-home assignment, one MR image (3D_WAND_T1_WEIGHTED.nrrd) and an associated label file (3D_WAND_T1_WEIGHTED-label.nrrd) were provided in the GBM folder. The label file provided a mask for the GBM that is present in the MRI.

Implementation¶

The code below performs the following tasks:

  • Separates the tumor region into Contrast enhancing and necrotic regions
    • Read image and label file
    • Smooth image using an edge-preserving filter (Curvature flow) for denoising
    • Apply tumor mask obtained from label file
    • On masked image, compute automated threshold to separate CE and NE regions
    • generate label image with background label 0, NE label 1, and CE label 2
    • write out image file
  • Computes volumes of these regions
    • Compute volumes and write out a CSV file

# function to show 3D SimpleITK image using matplotlib
def myshow(img, title=None, margin=0.05, dpi=80):
    nda = sitk.GetArrayViewFromImage(img)
    spacing = img.GetSpacing()
        
    if nda.ndim == 3:
        # fastest dim, either component or x
        c = nda.shape[-1]
        
        # the the number of components is 3 or 4 consider it an RGB image
        if not c in (3,4):
            nda = nda[nda.shape[0]//2,:,:]
    
    elif nda.ndim == 4:
        c = nda.shape[-1]
        
        if not c in (3,4):
            raise Runtime("Unable to show 3D-vector Image")
            
        # take a z-slice
        nda = nda[nda.shape[0]//2,:,:,:]
            
    ysize = nda.shape[0]
    xsize = nda.shape[1]
      
    # Make a figure big enough to accommodate an axis of xpixels by ypixels
    # as well as the ticklabels, etc...
    figsize = (1 + margin) * ysize / dpi, (1 + margin) * xsize / dpi

    fig = plt.figure(figsize=figsize, dpi=dpi)
    # Make the axis the right size...
    ax = fig.add_axes([margin, margin, 1 - 2*margin, 1 - 2*margin])
    
    extent = (0, xsize*spacing[1], ysize*spacing[0], 0)
    
    t = ax.imshow(nda,extent=extent,interpolation=None)
    
    if nda.ndim == 2:
        t.set_cmap("gray")
    
    if(title):
        plt.title(title)
    plt.show()
